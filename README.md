# 이가람

## 오늘 배운 건 오늘 적

영국 잉글랜드의 이튼 칼리지를 구경하던 한 관광객이 이렇게 완벽한 잔디밭을 어떻게 만들 수 있는지 정원사에게 물었다. 정원사는 **"그건 쉬워요."** 라고 대답했다. **"매일 아침 이슬을 털어주고, 이틀에 한 번 잔디를 깎아주고, 일주일에 한 번 잔디밭을 골라주면 되지요."**   
  
 "그게 다에요?" 관광객은 물었다.   
  
 "그게 답니다." 정원사는 대답했다. **"그렇게 500년만 하시면 당신 잔디밭도 이 정도로 괜찮아질 겁니다."**
